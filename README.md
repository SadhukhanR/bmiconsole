# Bmiconsole
Health Index Testing For Linux Users
## Installation
```bash
git clone https://github.com/SadhukhanR/bmiconsole.git
```
## Recommend Python Library
[Matplotlib](https://matplotlib.org) & [Numpy](https://numpy.org)

## Run
```sh
linux@linux~# bmiconsole 
```
